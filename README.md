# breve descrizione del lavoro svolto
in questo progetto tramite l'utilizzo di VS Code,ho creato uno script che contenesse i profili di una classe studenti.
in particolare, lo script permette di accedere ad informazioni come:
- nome, cognome, matricola dello studente,
- permette di aggiungere dei voti relativi agli esami degli studenti in un range da 18 a 30;
- calcola la media dei voti e la durata delle ore di studio.
- al termine poi effettua una breve presentazione degli studenti, con tutte le informazioni sopra citate.
